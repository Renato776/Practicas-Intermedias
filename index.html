GOOGLE VISION API

La API de Cloud Vision es un servicio que proporciona potentes funciones de reconocimiento de imágenes con una API simple. Puedes comprender y reconocer lo que hay en la imagen, permite el desarrollo de aplicaciones de próxima generación.

¿Cómo funciona Google Vision API?

FUNCIONALIDADES

  • Label/Entity detection, Escoge el objeto predominante en la imagen
  • OCR, lee y extrae el texto de las imágenes
  • Detección de contenido inapropiado
  • Detección facial
  • Detección de paisajes para detectar estructuras naturales o hechas por el hombre
  • Identificación de logos dentro de una imagen
